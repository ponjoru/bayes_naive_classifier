Подробное описание

Удаления стоп - слов из исходного текста.

Класс для удаления стоп - слов из вектора слов исходного текста.
Содержит одно поле данных, конструктор по умолчанию и три метода.

◆ removeWord()

void StopWords::removeWord ( vector< wstring > *  words)

Метод для удаления всех стоп - слов, которые встречаются в исходном тексте.
Внутри метода вызывается findWord(wstring word).
Если он вернет true, то слово удаляется из вектора всех слов, иначе вектор остается неизмененным

Аргументы
[in,out]words- указатель на вектор всех слов из исходного текста
