Подробное описание

template<typename string_typeT = std::wstring>
class stemming::stem< string_typeT >

Базовый класс для стеммера.

Класс реализует вспомогательные функции, такие как нахождение областей RV, R1, R2, поиск/удаление окончаний/суффиксов из различных областей слова, для алгоритма стеммера Портера.
